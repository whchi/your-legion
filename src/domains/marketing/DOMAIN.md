# Marketing Domain

Use this domain when the task involves market-facing communication, launch copy, positioning, audience messaging, campaign planning, landing page copy, announcement copy, or brand voice review.

Do not use this domain for internal-only code refactors, backend implementation with no user-facing messaging, pure financial analysis, or accounting treatment decisions.

Decisions:
- `decisions/brand-voice.md`

Examples:
